# MiniApps Emprende — Starter Kit v1.0

Plantilla oficial de infraestructura para todas las MiniApps Emprende.

## Inicio rápido

```bash
npm install
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000).

Documentación completa: [`docs/STARTER_KIT.md`](docs/STARTER_KIT.md).

## Qué incluye

- Next.js 16 + React + TypeScript
- Tailwind CSS v4 + shadcn/ui
- React Hook Form + Zod
- Lucide React + Poppins
- Layout base (Header / Main / Footer)
- Componentes reutilizables (Button, Input, Card, Section, PageContainer, EmptyState, ResultCard)
- Design system con colores de marca

## Qué no incluye

Login, dashboard, base de datos, APIs, IA ni funcionalidades de negocio. Este proyecto es solo plantilla.
